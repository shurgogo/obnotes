用CDN使用vue
<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
src可以直接另存为下载到本地
[[CDN]]
```html
<html>
<head>
  <meta charset="UTF-8">
  <title>初识vue</title>
  <script type="text/javascript" src="../js/vue.js">  </script>
  <!-- <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script> -->
</head>
<body>
</body>
</html>
```
